# Assets Directory

Place your application assets here.

## Directory Structure

```
assets/
├── images/
│   └── google_logo.png   (Optional - Google logo for sign-in button)
└── icons/
    └── (Any custom icons)
```

## Image Requirements

### Google Logo (Optional)
- **File**: `google_logo.png`
- **Size**: 48x48 px or higher (will be scaled to 24x24)
- **Format**: PNG with transparency
- **Source**: Download from Google Brand Resources or use Flutter's built-in icon

## Notes

- The app will work without custom assets
- If google_logo.png is missing, a default icon will be used
- All images should be optimized for mobile use
